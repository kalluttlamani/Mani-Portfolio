Project 2

Academic Performance Prediction Using Machine Learning Approaches: A Survey

Live Link

Project Overview

Accurately predicting students’ academic performance is essential for improving learning outcomes and supporting timely educational interventions. Effective forecasting enables educators to identify at-risk learners early and tailor instructional strategies to individual needs. Traditional approaches that rely on teachers’ intuition often lack precision, consistency, and scalability. Consequently, the past decade has witnessed a substantial increase in the adoption of machine learning and data mining techniques for academic performance prediction. However, no consensus has been reached regarding the most suitable algorithm for this task, as model performance varies across datasets, features, and learning contexts. This survey provides a comprehensive review of recent machine learning–based approaches for academic performance prediction. It examines the key features employed, algorithms applied, datasets utilized, and evaluation metrics adopted across existing studies. Unlike prior surveys, this work also analyzes relevant reviews published within the last ten years, comparing their methodologies, limitations, and contributions. Furthermore, we evaluate the performance of widely used machine learning models on popular open-access datasets to identify the best-performing algorithms in practical scenarios. All datasets and source codes used in this study are made publicly available to support reproducibility, future research, and algorithmic benchmarking.
